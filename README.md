## 🌟 Overview

**DigiPandit** is a modern digital platform that brings together **religious services, astrology consultations, guided rituals, devotional learning, online shopping, secure payments, and virtual experiences** into one unified ecosystem.

Instead of visiting multiple platforms for booking Pandits, consulting astrologers, purchasing Puja materials, learning rituals, or participating in online ceremonies, DigiPandit provides everything in a single integrated application.

The platform is designed for **users**, **Pandits**, **Astrologers**, and **administrators**, making religious services more accessible, transparent, and convenient while preserving traditional practices through modern technology.

---

# 🎯 Vision

To digitally transform the way people access religious services by providing a trusted platform that simplifies bookings, improves accessibility, preserves traditional knowledge, and connects verified experts with devotees worldwide.

---

# 💡 Problem Statement

Traditional religious services often face challenges such as:

- Difficulty finding verified Pandits
- Limited availability for online consultations
- Lack of transparent booking systems
- No centralized devotional learning platform
- Separate platforms for shopping, booking, and consultations
- Manual appointment management
- Limited accessibility for users living outside their hometown

DigiPandit solves these challenges by creating a unified ecosystem where every religious need can be managed from a single platform.

# 🚀 Getting Started

Follow the steps below to run DigiPandit locally.

---

## 📋 Prerequisites

Before getting started, ensure the following tools are installed:

- Node.js (v18 or later)
- npm / yarn
- MongoDB (Local or Atlas)
- Git

Optional Services

- Cloudinary
- Razorpay
- SMTP Service

---

# 📥 Clone Repository

```bash
git clone https://github.com/yourusername/digipandit.git

cd digipandit
```

---

# 📦 Install Dependencies

### Backend

```bash
cd backend
npm install
```

### Web

```bash
cd ../web
npm install
```

### Mobile

```bash
cd ../mobile
npm install
```

---

# ⚙ Environment Variables

Create `.env` files inside the required directories.

Example Backend Environment

```env
PORT=

MONGODB_URI=

JWT_SECRET=

JWT_EXPIRES_IN=

CLOUDINARY_CLOUD_NAME=

CLOUDINARY_API_KEY=

CLOUDINARY_API_SECRET=

RAZORPAY_KEY_ID=

RAZORPAY_KEY_SECRET=

EMAIL_USER=

EMAIL_PASSWORD=
```

---

# ▶ Running the Project

### Backend

```bash
cd backend

npm run dev
```

---

### Web

```bash
cd web

npm run dev
```

---

### Mobile

```bash
cd mobile

npm start
```
